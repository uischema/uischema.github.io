Link

A reference to another page or element

Properties

href URL The address of the page or element
openNew Open in new window/tab Whether to open this link in the same or in a new window/tab
text Text The text that is displayed

Definition

JSON 
{
    "@context": "http://uischema.org",
    "@type": "Link",
    "text": "string",
    "href": "string",
    "openNew": "bool"
}
Template 
<a data-ui-schema="uischema.org/Link" href="{{ href }}" {{# openNew }}target="_blank"{{/ openNew }}>{{ text }}</a>
