Link

A reference to another page or element

Properties

URL The address of the page or element
Open in new window/tab Whether to open this link in the same or in a new window/tab
Text The text that is displayed

Definition

JSON 
{
    "@context": "http://uischema.org",
    "@type": "Link",
    "text": "string",
    "href": "string",
    "openNew": "bool",
    "@i18n": {
        "en": {
            "@name": "Link",
            "@description": "A reference to another page or element",
            "href": {
                "@name": "URL",
                "@description": "The address of the page or element"
            },
            "openNew": {
                "@name": "Open in new window/tab",
                "@description": "Whether to open this link in the same or in a new window/tab"
            },
            "text": {
                "@name": "Text",
                "@description": "The text that is displayed"
            }
        }
    }
}
