Gallery

A collection of featured items

Properties

Heading The largest heading
Subheading The smaller heading
Body text The smaller text under the headings
Items The list of featured items

Options

Use squares Make the featured items square
Use cover links Make the featured items clickable
Is carousel Display all featured items in a single row
