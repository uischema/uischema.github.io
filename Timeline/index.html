Feature

An element for featuring other content

Gallery

A collection of featured items

Hero

A large element with an image background

Highlight

A side-by-side image and accompanying text

Image

A reference to an image

Link

A reference to another page or element

Timeline

A list of events

Timeline

A list of events

Parent: Gallery

Properties

items Items The events
link Link A link after the events

Definition

JSON 
{
    "@context": "http://uischema.org",
    "@type": "Timeline",
    "@topic": [ "media", "text" ],
    "@parent": "Gallery",
    "link": "Link"
}
Template 
<section data-ui-schema="uischema.org/Timeline" class="ui-timeline">
    <div class="ui-container">
        {{# heading }}
            <h2 class="ui-timeline__heading">{{ . }}</h2>
        {{/ heading }}
        
        {{# subheading }}
            <h3 class="ui-timeline__subheading">{{ . }}</h3>
        {{/ subheading }}
        
        {{# body }}
            <div class="ui-timeline__body">{{{ . }}}</div>
        {{/ body }}
        
        <div class="ui-timeline__items">
            {{# items }}
                <div class="ui-timeline__item">
                    {{> Feature }}
                </div>
            {{/ items }}
            
            {{# link }}
                <div class="ui-timeline__link">{{> Link }}</div>
            {{/ link }}
        </div>
    </div>
</section>
